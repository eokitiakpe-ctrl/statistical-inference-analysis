# Statistical Inference & Decision Analysis

This project explores hypothesis testing using binomial and normal models to analyze how p-values, likelihood ratios, and confidence intervals interact.

## Overview
- Evaluates statistical significance using real data scenarios  
- Compares p-values and likelihood ratios  
- Interprets confidence intervals in decision-making contexts  
- Demonstrates the difference between statistical and practical significance  

## Tools
Python (NumPy, SciPy, Matplotlib)

## Key Insight
Statistically significant results do not always imply strong evidence or meaningful real-world impact.
